This github repository contains all scripts which can be used for generating the textual notation of a SysML v2 model based of an input Excel file. 
It also contains the possibility to export a SysML v2 model to Excel which then can be used to import into Capella with the Python for Capella add-on.

Using either the master_import.py or the master_export.py the SysML v2 model can either be generated or extracted by the user who only has to specify the names of the files.
